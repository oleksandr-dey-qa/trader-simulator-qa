# 📋 Чекліст тестування самоконтролю для гри "Trader" (Spribe)

Цей чекліст розроблений для системного підходу до гри, мінімізації фінансових ризиків та контролю емоційного стану гравця під час сесії.

---

## 1. Підготовка перед стартом (Менеджмент)

- [ ] **Визначити банкрол:** Виділити на сесію суму, яку абсолютно не шкода втратити.
- [ ] **Встановити ліміт втрат (Stop-Loss):** Визначити суму, після програшу якої ви негайно закриваєте гру (наприклад, мінус 15% від банку).
- [ ] **Встановити ціль прибутку (Take-Profit):** Визначити фіксований відсоток прибутку, після якого сесія завершується (наприклад, плюс 20-30% до банку).
- [ ] **Розмір однієї інвестиції:** Сума однієї угоди не повинна перевищувати 1–3% від вашого поточного балансу.

## 2. Аналіз та Стратегія (Технічний мінімум)

- [ ] **Тест у Демо-режимі:** Провести мінімум 20-30 безкоштовних тренувальних раундів для ознайомлення з інтерфейсом та затримками (таймінгом).
- [ ] **Вибір тренду:** Поспостерігати за графіком без відкриття угод. Чи йде він стабільно в один бік (тренд), чи коливається на одному місці (флет)?
- [ ] **Вибір таймфрейму/таймінгу:** Визначити для себе комфортну тривалість угоди. Швидкі закриття мінімізують ризик раптового розвороту ринку.
- [ ] **Відмова від Мартингейлу:** Дати собі обіцянку не подвоювати кожну наступну ставку після програшу.

## 3. Під час гри (Контроль процесу)

- [ ] **Фіксація прибутку на низьких іксах:** Надійніше забирати невеликі множники (наприклад, x1.2–x1.5), ніж постійно чекати на «ікс-десять».
- [ ] **Пауза після серії невдач:** Якщо графік тричі поспіль пішов проти вашого прогнозу, зробити паузу на 3–5 хвилин.
- [ ] **Аналіз історії раундів:** Перевірити вкладку з попередніми результатами, щоб оцінити затяжність хвиль падіння чи зростання.

## 4. Психологічна гігієна (Захист від азарту)

- [ ] **«Тильт»-контроль:** Якщо з'явилося відчуття роздратування, злості або бажання «відігратися», негайно закрити гру.
- [ ] **Твереза голова:** Грати виключно у відпочилому стані. Алкоголь, втома чи стрес повністю вимикають раціональне мислення.
